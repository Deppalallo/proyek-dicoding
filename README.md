# Proyek Dicoding Collection Dashboard

## Setup environment
```
conda create --name main-ds python=3.11.5
conda activate main-ds
pip install pandas matplotlib seaborn jupyter streamlit babel
```

## Run steamlit app
```
streamlit run dashboard.py
```
